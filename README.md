# Programme Immobilier - Résidence Les Jardins

Application Next.js 14 pour la présentation d'un programme immobilier de 37 appartements.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Comptes de test

- **Admin**: admin@example.com / admin123
- **Client**: client@example.com / client123

## Structure

- `/app` - Pages Next.js (App Router)
- `/components` - Composants réutilisables
- `/data` - Données mockées (appartements et utilisateurs)
- `/lib` - Utilitaires (authentification)

## Fonctionnalités

- Authentification simple avec localStorage
- Dashboard avec grille de 37 appartements
- Page de détail pour chaque appartement
- Modal de contact avec validation
- Design moderne et minimal inspiré d'Aircall
